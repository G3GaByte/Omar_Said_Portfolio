## Description

Built an accurate WhatsApp interface mockup in SwiftUI, recreating core screens including the chat list, individual message threads, and message bubbles with sent/received states. The project focused purely on UI construction, replicating real-world design patterns using SwiftUI's native components like `List`, `ScrollView`, `HStack`, `VStack`, and custom `ViewModifier`s.

**What I learned**

Rebuilding a familiar, polished app from scratch highlighted the nuance behind layouts that look simple but require careful thought, things like dynamic bubble sizing, timestamp alignment, and scroll behavior. It also reinforced how to break a complex screen into small, reusable components and how to maintain visual consistency through a shared design system of colors, fonts, and spacing.

**How this improves my skillset**

Cloning a production-grade UI is one of the most effective ways to sharpen SwiftUI proficiency. This project built confidence in handling real-world layout challenges, custom styling, and component composition, skills that apply directly to any iOS role. It also developed an eye for design detail and taught me how to translate a visual reference into clean, maintainable SwiftUI code, which is critical when working from designer handoffs in a professional setting.logic cleanly separated from the view layer.
